เว็บสุ่มเมนูอาหารสำหรับมือถือ

วิธีเปิดใช้งานทันที
1. เปิดไฟล์ index.html ด้วยเบราว์เซอร์

วิธีทำเป็นลิงก์ฟรีด้วย Netlify
1. เข้าเว็บไซต์ Netlify และเข้าสู่ระบบ
2. ไปที่ Add new site > Deploy manually
3. ลากโฟลเดอร์ random-food-mobile หรือไฟล์ ZIP นี้ไปวาง
4. ระบบจะสร้างลิงก์เว็บไซต์ให้ทันที

หมายเหตุ
- เมนูโปรดจะถูกบันทึกในเบราว์เซอร์ของเครื่องนั้น
- ไม่ต้องใช้ฐานข้อมูลหรือเซิร์ฟเวอร์
